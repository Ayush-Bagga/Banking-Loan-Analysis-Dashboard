# Banking Loan Risk Analysis Dashboard

This project is a Power BI dashboard built to analyze banking loan data.

## Features
- Loan Overview Dashboard
- Risk Analysis Dashboard
- Bank Performance Dashboard
- Loan Affordability Calculator (What-if Analysis)

## Tools Used
- Power BI
- DAX
- Power Query

## Insights
- Loan approval and default trends
- Risk category distribution
- CIBIL score vs interest rate analysis
- Bank-wise performance comparison
